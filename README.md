# Blueprint OTU — Website

**Live site:** [otublueprint.com](https://otublueprint.com)

Built with React + TypeScript + Framer Motion + Vite.
Fonts: Clash Display + Syne + DM Mono via Google Fonts.

---

## Quick start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

---

## Deploy to GitHub Pages

```bash
npm run build
npm run deploy
```

This pushes the `dist/` folder to the `gh-pages` branch.
Go to **Settings → Pages → Source → gh-pages branch** to activate.
